# Cyberpatriot-Checklist-Team-ALM LINUX/UBUNTU/DEBIAN 1. README file 2. Read over the forensic questions and do them 3. Update entire system sudo apt update 4. Update Bash sudo apt-get dist-upgrade 5. Read through users on README and verify
  • No unnecessary users
  • No users not on the system
  • All users have correct permissions
  • Remove home folders of users you have deleted
6. sudo vi /etc/passwd to check for unauthorized users, or use Settings GUI. 7. Remove guest account sudo vi /etc/lightdm/lightdm.conf and add allow-guest = false to the end sudo restart lightdm may be required. 8. Check for autologin of unauthorized accounts sudo vi /etc/lightdm/lightdm.conf 9. Install and run lynis to detect threats with https://cisofy.com/download/lynis/ sudo lynis -c 10. sudo vi /etc/sudoers.d and check for threats from unauthorized users or utilities accessing sudo 11. Lock down root to disable su by sudo vi /etc/ssh/ssh_config and PermitRootLogin no also disable root for ssh if present. To lock down root, sudo passwd -l root 12. sudo vi /etc/group and remove unauthorized accounts 13.
